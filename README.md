# CommsCoach ROI Calculator

This repository contains a browser-based ROI calculator for CommsCoach built with Streamlit.

## Local run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Community Cloud
1. Push these files to a public GitHub repository.
2. Go to share.streamlit.io and sign in with GitHub.
3. Click **Deploy an app**.
4. Select your repository, branch, and `app.py` as the entrypoint.
5. Deploy.

## Files
- `app.py` - main Streamlit app
- `requirements.txt` - Python dependencies
